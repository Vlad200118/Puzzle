# Puzzle
Для того, чтобы игра запустилась, файлы с игрой необходимо распокавать из Puzzle.7z.
В архиве находятся:
    1. Папка "code" с кодами программы.
    2. Папки "font", "images" и "sounds", в которых располагаются шрифт, тайлы и звуки соответственно.
    3. .dll файлы, которые могут понадобиться для запуска .exe.
    4. .exe(win32) приложение.
    
Управление:
* ПКМ - выбор камня(фишки).
* ЛКМ - выбор ячейки, куда хотите переместить выбранный камень (выбранную фишку).

После правильной расстановки камней (фишек) игра останавливается (выбор фишек больше невозможен), окно с игрой можно только закрыть.
